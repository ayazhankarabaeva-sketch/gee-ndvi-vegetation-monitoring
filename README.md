# Automated NDVI / EVI Vegetation Monitoring

This project demonstrates an automated vegetation monitoring workflow built in Google Earth Engine using Sentinel-2 imagery.

The workflow performs:

• automatic scene selection around target dates  
• cloud filtering  
• NDVI and EVI calculation  
• vegetation zoning based on index thresholds  
• chart generation for field polygons  
• export links for visualization

Tools used:

- Google Earth Engine
- Sentinel-2 satellite imagery
- JavaScript scripting
